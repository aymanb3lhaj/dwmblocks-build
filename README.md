# dwmblocks-build

my simple build of dwmblocks

## Requirements

* Copy the scripts from my [dotfiles repo](https://github.com/aymanb3lhaj/dotfiles/tree/master/bin) into `$HOME/.local/bin/`

* Scripts:
  - date.sh
  - battery.sh
  - brightness.sh
  - volume.sh

## Installation

Cloning and compiling dwmblocks.

```
git clone <this-repo>
cd dwmblocks-build
make && sudo make clean install
```
